# docs/

This folder holds screenshots and demo assets for the README.

## Needed

- `demo.gif` - 20-30 second screen recording showing Claude answering "what am I working on?" after a fresh session start with no context file loaded
- `install-success.png` - terminal screenshot of a successful `setup.py` run
- `mempalace-status.png` - screenshot of `mempalace_status` output showing indexed entries

## Recording the demo GIF

1. Complete a fresh setup in a clean folder
2. Open Claude Code, start a new session
3. Ask: "What am I currently working on?"
4. Show Claude responding with content from `Brain/identity.md` or `Brain/Projects/`
5. Record with any screen capture tool that exports GIF (LICEcap, ShareX, Kap on Mac)
6. Keep it under 30 seconds and under 5MB
